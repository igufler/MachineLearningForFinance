## About the Course

This is a Master's-level course on applying machine learning methods to financial problems. The core objective is to build a working understanding of the main ML techniques — their assumptions, their strengths, and their limitations when applied to real financial data.

By the end of the course, you should be able to:

- choose an appropriate method for a given financial question,
- implement it correctly,
- interpret the results honestly.



## Topics

| Block | Content |
|---|---|
| Introduction & KNN | ML framework, bias–variance trade-off, flexibility |
| OLS | Regression, inference, extensions |
| Classification | Logit, Multiple Logit, Multinomial Logit |
| Resampling | Cross-validation, walk-forward, bootstrap |
| Model Selection & Regularization | Subset selection, Ridge, Lasso, Elastic Net |
| Dimension Reduction | PCA, PLS, factor interpretation |
| Tree-Based Methods | Decision trees, random forests, boosting |
| Neural Networks | Regularization, RNNs/LSTMs |
| Reinforcement Learning | MDP, Q-learning, Deep QL, portfolio applications |
